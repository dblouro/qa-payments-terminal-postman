# qa-payments-terminal-postman

PT
Mini‑projecto de Quality Assurance que simula um backend de pagamentos para terminais (TPA) usando um mock server Postman. Inclui testes funcionais de pagamento, cancelamento, consulta de estado e exemplos de defeitos, com documentação pensada para vaga de QA em ambiente financeiro.

EN
Mini‑Quality Assurance project that simulates a payments backend for terminals (TPA) using a Postman server mock. Includes functional tests of payment, cancellation, status consultation and examples of defects, with documentation designed for a QA position in a financial environment.
